# TicTacToe

Dieses Java-Projekt ist ein einfaches TicTacToe-Spiel für die Konsole.

Es gibt zwei Spieler. Spieler 1 spielt mit `X` und Spieler 2 mit `O`. Die Spieler wählen abwechselnd ein Feld von 1 bis 9 aus. Das Programm prüft automatisch, ob ein Spieler gewonnen hat oder ob es ein Unentschieden gibt.

## Klassen

* `Main` startet das Spiel.
* `TicTacToe` enthält das Spielfeld, die Eingabe, den Spielerwechsel und die Gewinnprüfung.

## Starten

In IntelliJ einfach die Datei `Main.java` starten.
